이 플러그인은 FaigerSYS에 의해 작성되었습니다.
깃허브 페이지 : https://github.com/FaigerSYS/MapImageEngine

만약, 당신이 버그를 발견한다면, 제가 알도록 해주세요! 새 이슈를 깃허브에 작성해주세요.

자신의 이미지 제작 법:
	1. 아무 이미지 파일을 변환하세요. ('png', 'jpg', 기타 등등...)
		- MIE_Convert 플러그인을 이용하세요. : https://github.com/FaigerSYS/MapImageEngine/tree/master/MIE_Converter
		- 온라인 변환기를 이용하세요 (플러그인 미작동시) : http://mie-converter.ml
	1.1 당신은 청크 이미지 크기를 128x128로 바꿀 필요가 없습니다. 변환기가 크기를 바꿔줄 것입니다.
	1.2 당신은 청크 이미지를 필요로 하지 않습니다. 블럭들의 크기를 선택해주세요.
	2. 변환된 이미지 파일을 'images' 폴더에 넣어주세요.
	3. 끝! 이제 너는 이 이미지를 사용할 수 있습니다.
	
어떻게 이미지를 설치방법:
	1.'/mie list' 를 사용해 사용 가능한 이미지 파일을 찾아주세요.
	2. 그것들 중 하나를 선택해서 '/mie place <이미지 이름>' 를 쳐주세요.
	2.1 이미지의 쉬운 설치를 위해 깃발들을 사용할 수 있습니다. ( ('/mie place') 로 얻은 모든 문자열을 입력하는 것이 필요합니다.)
	2.2 '/mie exit' 으로 설치모드를 떠나세요.
	3. 설명에 따르고 사진은 설치될 것입니다.
	
명령어의 모든 복록을 보기위해서, '/mie' 를 입력해주세요.

행운을 빌어요!
